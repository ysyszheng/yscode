YSCODE
=====
A tiny Code Editor

![YSCODE](./assets/img/demo.png)

Env
-----
```bash
# Use conda to create a virtual environment
$ conda create -n <env> --file requirements.txt
$ conda activate <env>
# or use pip
$ pip install PyQt5 
```

Run
-----
```bash
$ python3 ./src/main.py
```
